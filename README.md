# pavear
이 이야기는 순전한 허구입니다. 파베어라는 나라는 존재하지 않으며, 싸이닉이라는 도시도, 아마라는 여행자도 실재하지 않습니다. 다만 그 안에서 벌어지는 일들이 어딘가 낯익다면, 그건 독자의 몫입니다. This is a work of fiction. The nation of Pavear does not exist, nor does the city of Cynic, nor the traveler Ama. However, if what unfolds within feels strangely familiar — that is for the reader to decide.
